# Waterfall 瀑布流

## 参数

| 参数    | 说明   | 类型   | 可选值 | 默认值 |
| ------- | ------ | ------ | ------ | ------ |
| column  | 列数量 | number |        | 2      |
| gutter  | 间隔   | number |        | 20     |
| nodeKey | 匹配值 | string |        | id     |

## 方法

| 事件名称 | 说明     | 参数     |
| -------- | -------- | -------- |
| refresh  | 刷新列表 | function |
| append   | 追加数据 | function |
| clear    | 清空数据 | function |
| update   | 更新数据 | function |
